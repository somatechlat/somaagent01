SomaAgent01 System Configuration

Version 2.1 • Last updated: December 22, 2025

1. Infrastructure Overview

This document outlines the complete system configuration for the SomaAgent01 deployment, including server infrastructure, model configurations, and integration settings.

  • Primary API Gateway: Port 8020 (Django Ninja)
  • WebSocket Server: Port 8021 (Django Channels)
  • SomaBrain Memory: Port 8030
  • AgentVoiceBox: Port 8035

2. Agent Modes

The system supports six operational modes, each with specific permission levels:

  • STD - Standard user operations
  • TRN - Training mode with cognitive parameter access
  • ADM - Full administrative control
  • DEV - Developer mode with debug access
  • RO - Read-only mode
  • DGR - Degraded mode (failover)
