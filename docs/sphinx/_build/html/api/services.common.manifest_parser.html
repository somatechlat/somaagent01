services.common.manifest_parser module

Manifest ingestion pipeline for capsule YAML files.

Parses manifest.yaml files and normalizes them to CapsuleDefinition records. Validates policy, risk, and HITL fields. Rejects incompatible capsules based on environment mode (DEV, STAGING, PROD).
