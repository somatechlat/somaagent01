class admin.core.helpers.secrets.StreamingSecretsFilter(key_to_value, min_trigger=3)[source]

Bases: object

Stateful streaming filter that masks secrets on the fly.

  • Replaces full secret values with placeholders §§secret(KEY) when detected.

  • Holds the longest suffix of the current buffer that matches any secret prefix (with minimum trigger length of 3) to avoid leaking partial secrets across chunks.

  • On finalize(), any unresolved partial is masked with ‘***’.
