services.multimodal.base_provider module

Base class for multimodal providers.

Defines the abstract interface that all multimodal providers must implement, including generation, validation, and cost estimation methods.

SRS Reference: Section 16.6 (Execution Engine) Feature Flag: SA01_ENABLE_multimodal_capabilities
